## ğŸš€ Getting Started

### Prerequisites

- **C Compiler**: GCC, Clang, or MinGW
- **Text Editor/IDE**: VS Code, Code::Blocks, Dev-C++, or Vim
- **Basic Terminal/Command Line Knowledge**

### Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Vortex-ParthAjmera/C-programming.git
   cd C-programming
   ```

2. **Navigate to a Specific Directory**
   ```bash
   cd ARRAYS
   # or
   cd RECURSION
   ```

3. **Compile a Program**
   ```bash
   gcc program_name.c -o program_name
   # or
   gcc program_name.c -o program_name.exe  # Windows
   ```

4. **Run the Compiled Program**
   ```bash
   ./program_name        # Linux/Mac
   program_name.exe      # Windows
   ```

## ğŸ”§ Compilation & Execution Guide

### Using GCC (Linux/Mac/Windows with MinGW)

```bash
# Basic compilation
gcc program.c -o program

# With warnings enabled (recommended)
gcc -Wall -Wextra program.c -o program

# With debugging symbols
gcc -g program.c -o program

# Optimized compilation
gcc -O2 program.c -o program

# Run the program
./program
```
